Comparative Research for ISEA 2017

Needfindings

The needs are based on how people purchase their grocery

1. User needs information on the items they purchased.
2. User needs faster method to access information.
3. User will need to know how valid the information is on the product.
3. User needs know the price.
4. User needs scan the barcode.
5. User uses phone to look up information online.
6. User likes to make comparison.
7. User needs to have someone or someplace to ask questions.
8. User likes to complete their main task "shopping".
9. User does not like to be bothered.
10.User needs clear instruction.
