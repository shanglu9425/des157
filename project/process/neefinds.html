Comparative Research for ISEA 2017

Needfindings

  • User sometime likes to skip instruction
  • User does not like spend time during log in or sign up process
  • User likes to explore function by himself/herself.
  • User needs time to figure out different functions.
  • Description for products is important to user.
  • User relies on reviews and ratings to judge a item.
  • User likes attractive icons and graphics.
  • User likes to skip uncessary information or information that he/she cannot understand.
  • User needs image
  • User needs some kind of guide to help when he/she gets lost.
