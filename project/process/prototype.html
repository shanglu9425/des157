Comparative Research for ISEA 2017

Needfindings

The user inititally had troubles understanding the prototype because I did not label each page and the function of the buttons. Therefore, the user has no guidance to rely on. Although I included arrows to demonstrate how the application works and flow, but it still seemed more descrption was needed. In second prototype, I used letter labelings, and I provided more details for the user to make reference. I also specified the functions of each button, so the user will know what each button does, and what will they trigger.
